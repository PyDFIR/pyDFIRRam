## Quick installation

### Prerequisites 
- Python
- Poetry (for dev)
- pip

### From source
On a standard Linux distribution :
```shell
git clone https://github.com/pydfir/pydfirram
poetry shell
poetry install
```
### From pip stable

```shell
pip install pydfirram
```
### From pip dev

```bash
pip install -i https://test.pypi.org/simple/ pydfirram
```
